#### 个人谨记:

1.ord(c)是获取字符c的ASCII码值。ord('a')返回97，ord('A')返回65，ord('0')返回48。

2.生成器表达式是类似于列表推导式，但返回的是一个生成器对象，而不是一次性构建整个列表。
(c == 0 for c in cnt)
(expression for item in iterable if condition)
圆括号 () 表示这是一个生成器表达式（如果用作函数参数，可以省略外层括号，但通常保留以清晰表达）。
expression 是对每个元素计算的表达式，这里就是 c == 0（返回布尔值）。
for c in cnt 是迭代部分，依次取出 cnt 中的每个元素赋值给 c。
可选的 if 子句用于过滤，这里没有使用。

3.all() 是 Python 的内置函数，接受一个可迭代对象作为参数，并返回一个布尔值：
如果可迭代对象中所有元素都为真（或可迭代对象为空），则返回 True，否则返回 False。
all() 具有短路求值行为。一旦遇到第一个假值，就会立即返回 False，不再继续迭代剩余元素。这可以节省不必要的计算。
