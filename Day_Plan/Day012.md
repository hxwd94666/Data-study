## ⏰ 详细执行日程与学习闭环 (Execution Plan)

### 🌅 时间块 1：Python 面向对象与类的封装 (1.5 Hours)
* **【学什么】**：什么是类 (Class) 与对象 (Object)；构造函数 `__init__` 的作用；极其关键的 `self` 指针到底代表什么。
* **【怎么学】**：B 站搜索“黑马 Python 面向对象基础”。不要深究理论，重点看代码里怎么把零散的变量封装进一个“壳子”里。
* **【去哪练】**：在 PyCharm 本地环境中实操。
* **【达标效果与资产沉淀】**：在 `First_month/` 目录下新建 `day12_oop_basics.py`。要求：定义一个 `DataPipeline` 类，包含 `source_db` 和 `target_db` 属性，以及一个 `start_extract()` 方法。实例化该类并调用方法。必须包含类级别的 Google Docstring。

### ☀️ 时间块 2：算法突围 - 栈 (Stack) 与闭环校验 (1.5 Hours)

* **【学什么】**：栈“后进先出 (LIFO)”的底层思想，以及如何在 Python 中用列表 (List) 模拟栈的 `append()` 和 `pop()`。
* **【怎么学】**：访问“代码随想录”网站，搜索“20. 有效的括号”。理解为什么这种对称匹配问题必须用栈来解决。
* **【去哪练】**：LeetCode 第 20 题（有效的括号）。
* **【达标效果与资产沉淀】**：在 LeetCode 网页端纯盲打通过后，将代码沉淀至 `First_month/day12_valid_parentheses.py`。严禁使用复杂的正则，必须用栈结构。

### 🗄️ 时间块 3：SQL 跃迁 - Hive DDL 与数仓建表规范 (1 Hour)
* **【学什么】**：HiveSQL 数据定义语言 (DDL)。重点死磕 `CREATE EXTERNAL TABLE` 与 `PARTITIONED BY` 语法。
* **【怎么学】**：B 站搜索“Hive 内部表 外部表 区别”。理解为什么在生产环境中，原始数据层 (ODS) 绝对只能建外部表！
* **【去哪练】**：本地 PyCharm 新建 SQL 文件进行建表语句的盲打默写。
* **【达标效果与资产沉淀】**：在 `First_month/day12_hive_ddl.sql` 中编写一段企业级 ODS 层外部表的建表语句。包含：字段注释 (`COMMENT`)、分区 (`PARTITIONED BY (dt string)`)、以及行格式 (`ROW FORMAT DELIMITED`)。关键字强制全大写。
