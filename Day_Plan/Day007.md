## ⏰ 详细执行日程与学习闭环 (Execution Plan)

### 🌅 时间块 1：Python 工程化防线 - 异常处理 (09:00 - 11:30)
* **【学什么】**：`try...except...finally` 机制的执行流，以及常见内置异常类型（如 `ZeroDivisionError`, `KeyError`, `ValueError`）。
* **【怎么学】**：B 站搜索“Python 异常处理 黑马”或查阅官方文档。理解 `finally` 为什么无论是否报错都一定会执行（通常用于关闭数据库连接）。
* **【去哪练】**：在 PyCharm 本地环境进行断点调试与主动抛错（`raise`）测试。
* **【达标效果与资产沉淀】**：在 `First_month/` 目录下新建 `day07_exceptions.py`。要求：模拟一个数据量比对函数，人为制造一个除数为 0 的异常，并用 `try-except` 捕获拦截，在 `finally` 中打印“巡检结束”。必须包含完整 Google Docstring。

### ☀️ 时间块 2：算法突围 - 字符串与双指针 (14:00 - 15:30)
* **【学什么】**：利用双指针在原地（O(1) 空间复杂度）修改字符串/字符数组。
* **【怎么学】**：访问“代码随想录”网站，搜索“344. 反转字符串”。观察左右指针如何向中间逼近并完成元素交换。
* **【去哪练】**：LeetCode 第 344 题（反转字符串）。
* **【达标效果与资产沉淀】**：放弃 Python 切片语法糖（如 `s[::-1]`），强制使用双指针手写交换逻辑。在 LeetCode 盲打通过后，沉淀至 `First_month/day07_reverse_string.py`。

### 🗄️ 时间块 3：SQL 跃迁 - 空值清洗与防倾斜 (15:30 - 17:30)
* **【学什么】**：处理脏数据的基石函数：`COALESCE()`（返回第一个非空值）以及 `NVL()`/`IFNULL()`。
* **【怎么学】**：B 站搜索“SQL COALESCE 空值处理”。理解在进行表 JOIN 或者算术计算时，为什么 NULL 值会导致整个结果变成 NULL，以及如何用 `COALESCE` 赋默认值兜底。
* **【去哪练】**：本地 PyCharm 新建 SQL 文件进行逻辑默写，或在牛客网 SQL 题库寻找“空值处理”相关题目验证。
* **【达标效果与资产沉淀】**：在 `First_month/day07_null_handling.sql` 中编写 3 段对比代码，分别展示 `COALESCE` 处理多字段空值、以及在 `SUM()` 聚合前过滤 NULL 值的场景。关键字强制全大写。
