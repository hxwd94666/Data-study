### ⏰ Day 1 详细执行日程与实现指南

#### 🌅 上午：指法重塑与代码破冰（9:00 - 11:30）

- **09:00 - 09:40 | ⌨️ 打字重塑与第一战（极其关键的40分钟）**
  - **方法论（必读）**：
    1. 寻找“盲文点”：你的双手食指必须放在键盘带有小凸起的 `F` 和 `J` 键上。其余手指自然平放在 `ASDF` 和 `JKL;` 上。这叫“基准键（Home Row）”。
    2. **铁律**：从今天起，**哪怕一个字母找了10秒钟，也绝对不许低头看键盘！** 哪怕刚开始一分钟只能打 10 个字，也要忍住看键盘的冲动。这是建立肌肉记忆的唯一途径。
  - **学习资源**：打开网站 **Keybr.com**（极其推荐，专治各种指法不服，会根据你的弱点智能出题）或 **TypingClub.com**。
  - **行动**：进行 30 分钟的沉浸式盲打练习。体会手指去够其他按键然后再退回基准键的感觉。
- **09:40 - 10:20 | SQL 手感唤醒（减量为 2 道题）**
  - **学习资源**：[牛客网 (nowcoder.com)](https://www.nowcoder.com/) -> 在线编程 -> “SQL必知必会”。
  - **行动**：挑前 2 道最简单的题（`SELECT` 和 `WHERE`）。**强迫自己用标准指法慢慢敲入 `SELECT` 这些关键字**。即使很慢也不要急，当做打字练习。把通过的代码保存在本地 `day01_sql_practice.sql` 中。
- **10:20 - 11:30 | 算法首战：二分查找（规范大于速度）**
  - **学习资源**：力扣搜索 **第704题（二分查找）**，思路看《代码随想录》网站的图解。
  - **行动**：在本地新建 `day01_binary_search.py`。用**规范的 Type Hint（类型提示）和注释**写出这道题。不要管写得多慢，哪怕只敲这一道题敲了 40 分钟，只要你是用盲打姿势敲出来的，今天就赢了。

#### ☀️ 下午：IDEA 兵工厂与 Python 入门（14:00 - 17:30）

- **14:00 - 15:30 | IDEA 环境与基建**
  - **学习资源与行动**：
    1. **Python 安装**：官网下载 Python 3.10+，**务必勾选 “Add Python to PATH”**。
    2. **IDEA 配置**：安装并打开 IntelliJ IDEA（社区版免费够用，有旗舰版更好）。打开后，进入 `Plugins` (插件市场)，搜索 **“Python”**（如果是社区版，搜 Python Community Edition）并安装，重启 IDEA。
    3. **Git 安装**：官网下载 Git，一路 Next。
    4. **账号注册**：搞定 GitHub 账号注册。
- **15:30 - 17:30 | Python 基础语法（慢练）**
  - **学习资源**：B站搜索“黑马程序员 Python”，跳过环境安装，直接看“变量类型”、“if 判断”、“for/while 循环”、“列表 List”。
  - **行动**：在 IDEA 里新建一个 Python Project，命名为 `180Days-Data-Offer`。新建一个文件 `day01_python_basics.py`。一边看视频，一边用你那“极其缓慢但正确”的指法跟着敲循环和列表切片的代码。

------

#### 🌙 晚上：Markdown 复盘与 GitHub 首推（19:00 - 21:00）

- **19:00 - 19:40 | Markdown 极速入门与慢速书写**
  - **学习资源**：搜索“菜鸟教程 Markdown”。
  - **行动**：在 IDEA 项目根目录下新建 `README.md`。今天你的打字速度注定了你写不了长篇大论。简明扼要地写几句话（包含 `##` 标题和 `-` 列表即可），记录一下今天强忍着不看键盘的痛苦与收获。
- **19:40 - 20:30 | 见证奇迹的 Git 提交**
  - **学习资源**：其实 IDEA 内部集成了极度强大的 Git 可视化工具，但作为初学者，**建议先学命令行**。
  - **行动**：打开 IDEA 底部的 `Terminal`（终端）标签页。依次输入：
    1. `git init`
    2. `git add .`
    3. `git commit -m "Day 1: 规范打字姿势，配置IDEA，完成二分查找"` （慢慢敲这些命令）
    4. 去 GitHub 网页新建仓库，把给你的那两行 `git remote add...` 和 `git push...` 复制到终端执行。
- **20:30 - 21:00 | 今日复盘与缓冲**
  - 刷新 GitHub 网页，看绿方块是否点亮。
  - 如果今天因为打字慢导致某些任务没做完，**不要气馁，直接顺延到明天，绝对不要为了赶进度而换回“双指禅”！**
